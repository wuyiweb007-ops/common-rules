# UI 设计规范文档

## 1. 设计风格

### 1.1 视觉风格
- **定位**：现代年轻风
- **色彩基调**：渐变色系
- **主色**：#667EEA（紫蓝） → #764BA2（深紫）渐变
- **辅助色**：#F093FB → #F5576C（粉红渐变，用于强调）
- **中性色**：
  - 背景：#FFFFFF / #F8F9FA
  - 文字主色：#1A202C
  - 文字次要：#718096
  - 边框：#E2E8F0
- **圆角**：统一 12px（卡片）、8px（按钮）、16px（弹窗）

### 1.2 字体规范
- **标题**：24px / Bold
- **副标题**：18px / Semibold
- **正文**：16px / Regular
- **辅助文字**：14px / Regular
- **按钮文字**：16px / Medium
- **字体**：系统默认（-apple-system, BlinkMacSystemFont, "Segoe UI"）

---

## 2. 页面结构（移动优先）

### 2.1 布局方式
- **单栏垂直布局**
- **最大宽度**：750px（超出后居中显示）
- **内边距**：左右 20px
- **组件间距**：24px

### 2.2 页面顶部固定元素
- Logo（左上角，40px 高度）
- 合规声明按钮（右上角，"？"图标）

---

## 3. 首页（分析页面）

### 3.1 页面元素顺序
1. 顶部标题区
2. 场景选择区
3. 照片上传区
4. 开始分析按钮
5. 分析结果区（默认隐藏）

### 3.2 顶部标题区
- **标题**："发现你的第一印象"
- **副标题**："AI 帮你看懂那张照片"
- **样式**：居中对齐，渐变文字

### 3.3 场景选择区
- **UI 形态**：3 个横向卡片
- **卡片结构**：
  - 图标（48px）
  - 场景名称（16px）
  - 简短说明（14px，灰色）
- **选中状态**：渐变边框 + 背景色浅化
- **未选中**：白色背景 + 灰色边框
- **图标映射**：
  - job_interview：💼
  - dating：💕
  - business_profile：✨

### 3.4 照片上传区
- **默认状态**：虚线边框矩形（4:3 比例）
- **内容**：相机图标 + "点击上传照片"
- **已上传状态**：显示缩略图 + 右上角删除按钮
- **限制提示**：底部显示"≤5MB，支持 JPG/PNG/HEIC"

### 3.5 开始分析按钮
- **宽度**：100%
- **高度**：48px
- **样式**：渐变背景（主色）
- **文字**："开始分析"
- **禁用状态**：未选场景或未上传照片时置灰

### 3.6 分析进度状态
- **加载动画**：3 个跳动的圆点（渐变色）
- **进度文案**：
  - "正在识别人脸..." (1s)
  - "正在分析第一印象..." (2-4s)
  - "正在生成建议..." (1-2s)

---

## 4. 分析结果区

### 4.1 结果展示顺序（逐条展开动画）
1. 第一印象摘要（免费）
2. 正向信号（付费）
3. 潜在风险（付费）
4. 优化建议（部分免费）

### 4.2 动画规则
- **展开动画**：从上到下，每条间隔 200ms
- **进入效果**：淡入 + 向上滑动（translateY: 20px → 0）
- **时长**：300ms

### 4.3 第一印象摘要（免费）
- **卡片样式**：白色背景 + 阴影
- **标题**："第一印象"（18px / Semibold）
- **内容**：3 行文字（16px / Regular）
- **图标**：✨（渐变色）

### 4.4 付费内容遮罩（正向信号 + 潜在风险）
- **遮罩方式**：模糊（blur(8px)）+ 半透明白色蒙版
- **卡片高度**：固定 120px
- **锁定图标**：居中显示 🔒
- **提示文字**："解锁查看完整分析"

### 4.5 优化建议（部分免费）
- **免费展示**：前 2 条
- **付费展示**：完整列表（5-7 条）
- **每条结构**：
  - 圆形图标（32px，渐变背景）
  - 建议标题（16px / Medium）
  - 建议详情（14px / Regular，灰色）
- **付费部分**：第 3 条开始模糊遮罩

### 4.6 解锁按钮
- **位置**：结果区底部
- **宽度**：100%
- **高度**：48px
- **样式**：粉红渐变背景
- **文字**："解锁完整分析 ¥9.9"

### 4.7 分享按钮（付费后显示）
- **位置**：解锁按钮下方
- **样式**：白色背景 + 渐变边框
- **文字**："分享我的分析结果"

---

## 5. 付费弹窗（底部弹起）

### 5.1 弹窗样式
- **形态**：底部弹起（translateY: 100% → 0）
- **圆角**：顶部 16px
- **背景**：白色
- **遮罩**：半透明黑色（rgba(0,0,0,0.5)）

### 5.2 弹窗内容结构
1. 顶部拖拽条（灰色圆角矩形）
2. 标题："解锁完整分析"
3. 价格："¥9.9"（32px / Bold，渐变色）
4. 收款码（240px 正方形）
5. 提示文字："扫码支付后点击下方按钮"
6. "已支付"按钮（渐变背景，48px 高度）
7. 关闭按钮（右上角 ✕）

### 5.3 动画时长
- 弹起：300ms（ease-out）
- 关闭：200ms（ease-in）

---

## 6. 对比分析页面

### 6.1 照片上传区（左右分屏）
- **布局**：两列等宽（左右各 50%，中间 8px 间距）
- **卡片结构**：
  - 标签（"照片 A" / "照片 B"）
  - 虚线边框矩形（1:1 比例）
  - 已上传：显示缩略图 + 删除按钮

### 6.2 对比结果展示
- **优胜照片**：左右两张并排，优胜方加粗边框（渐变）
- **判断标题**："照片 A 在该场景下更合适"
- **判断依据**：3 条理由（每条带圆点图标）
- **使用建议**：单独卡片展示

---

## 7. 分享页面

### 7.1 页面结构
- 顶部：品牌标识（Logo + Slogan）
- 中部：完整分析结果（同主页结构）
- 底部：体验按钮（"我也要分析" → 跳转首页）

### 7.2 样式差异
- 无遮罩（全部可见）
- 无解锁按钮
- 底部固定"体验"按钮（粉红渐变）

---

## 8. 交互反馈

### 8.1 按钮反馈
- **点击**：scale(0.95) + 透明度 0.8
- **时长**：100ms

### 8.2 错误提示
- **形态**：顶部 Toast（红色背景）
- **时长**：3 秒自动消失
- **内容示例**：
  - "未检测到人脸，请重新上传"
  - "照片大小超过 5MB"
  - "分析超时，请稍后重试"

### 8.3 成功提示
- **形态**：顶部 Toast（绿色背景）
- **时长**：2 秒自动消失
- **内容示例**："支付确认成功"

### 8.4 加载状态
- **按钮加载**：文字替换为旋转动画
- **禁用**：不可点击 + 透明度 0.6

---

## 9. 合规声明弹窗

### 9.1 触发方式
- 点击右上角"？"图标

### 9.2 弹窗样式
- **形态**：居中模态框
- **宽度**：80%（最大 400px）
- **圆角**：16px
- **内容**：
  - 标题："使用说明"
  - 正文：spec 中的合规声明
  - 关闭按钮："我知道了"

---

## 10. 响应式适配

### 10.1 移动端（≤750px）
- 单栏布局
- 内边距 20px
- 对比分析左右分屏（各 50%）

### 10.2 桌面端（>750px）
- 内容居中，最大宽度 750px
- 两侧留白背景
- 对比分析依然左右分屏

---

## 11. 性能优化

### 11.1 图片处理
- 上传前压缩至 1080px 宽度
- 缩略图展示使用 object-fit: cover

### 11.2 动画性能
- 使用 transform 和 opacity（GPU 加速）
- 避免 width/height 动画

---

## 12. 微信浏览器适配

### 12.1 特殊处理
- 禁用长按保存图片（收款码除外）
- 禁用下拉刷新
- 禁用字体缩放

### 12.2 分享优化
- 配置微信分享卡片（标题 + 描述 + 缩略图）

1. **按钮**：主要按钮 / 次要按钮 / 文字按钮

3. **Toast**：成功 / 错误 / 提示

5. **加载动画**：跳动圆点 / 旋转
6. **图片上传**：单张 / 对比双张
